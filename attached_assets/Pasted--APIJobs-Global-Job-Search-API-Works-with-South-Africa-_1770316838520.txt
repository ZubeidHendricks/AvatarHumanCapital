Search engine with filters and ranking

Matches candidates to jobs or search queries

what is open source 

Short and clean:

Open source means the source code is publicly available for anyone to view, use, modify, and share.

Instead of software being a “black box” owned and controlled by one company, open-source projects are built in the open by communities, companies, or individuals.

🔓 What makes something open source?

For software (and sometimes data or models) to be truly open source, it usually means:

✅ You can see the source code

✅ You’re allowed to modify it

✅ You can redistribute it

✅ It’s licensed under an open-source license

Common licenses:

MIT

Apache 2.0

GPL

BSD

🧠 Simple analogy

Closed source = Buying a car with the hood welded shut

Open source = You can open the hood, tune the engine, replace parts, and even build your own version

Open source ≠ free data

Software can be open source

Job boards sell access

Candidate databases are closed

Executive data is licensed, not shared
