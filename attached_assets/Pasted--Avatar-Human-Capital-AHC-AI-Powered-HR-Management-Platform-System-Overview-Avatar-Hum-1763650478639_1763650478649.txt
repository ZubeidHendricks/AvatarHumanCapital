# Avatar Human Capital (AHC) - AI-Powered HR Management Platform

## System Overview

Avatar Human Capital is a comprehensive, AI-powered Human Resources Management platform designed to automate and optimize the entire employee lifecycle—from recruitment to ongoing HR management. Built with cutting-edge AI agentic workflows, the system provides intelligent automation for HR processes while maintaining human oversight and control.

## System Architecture

### Technology Stack

#### Frontend
- **Framework**: Next.js 16 with Turbopack
- **UI Library**: Fluent UI React Components
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Query (TanStack Query)
- **Deployment**: Vercel

#### Backend
- **Framework**: FastAPI (Python)
- **Database**: PostgreSQL (DigitalOcean Managed Database)
- **AI/ML**: Groq LLaMA 3.1 70B, LangChain, LangGraph
- **File Storage**: Local filesystem with cloud integration
- **Deployment**: DigitalOcean App Platform

#### Infrastructure
- **Database Host**: aihr-postgres-do-user-23094081-0.g.db.ondigitalocean.com
- **Backend API**: aihr-backend-hmew5.ondigitalocean.app
- **Frontend**: Deployed on Vercel
- **Version Control**: GitHub

### LangGraph Workflow Engine

The system uses LangGraph to orchestrate complex multi-agent workflows:

```python

├── Job Posting Agent
├── Candidate Screening Agent
├── Interview Scheduling Agent
├── Background Check Agent
├── Onboarding Agent
└── HR Operations Agent
```

## API Endpoints

### Base URL
- Production: `https://aihr-backend-hmew5.ondigitalocean.app/api/v1`
- Local: `http://localhost:8000/api/v1`

### Authentication
```
POST /auth/login
POST /auth/register
POST /auth/refresh
GET  /auth/me
```

### Jobs
```
GET    /jobs/
POST   /jobs/
GET    /jobs/{id}
PUT    /jobs/{id}
DELETE /jobs/{id}
```

### Candidates
```
GET    /candidates/
POST   /candidates/
GET    /candidates/{id}
PUT    /candidates/{id}
DELETE /candidates/{id}
POST   /candidates/upload-resume
```

### Interviews
```
GET    /interviews/
POST   /interviews/
GET    /interviews/{id}
PUT    /interviews/{id}
DELETE /interviews/{id}
```

### Workflow
```
POST /workflow/complete-recruitment-flow
GET  /workflow/workflow-status/{job_id}
POST /workflow/screen-candidates
POST /workflow/create-interviews
```
